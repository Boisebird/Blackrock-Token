Here’s how you can launch your Black Rock website for free using GitHub Pages or Netlify:

Option 1: GitHub Pages (Free & Easy)

If you’re okay with a GitHub-branded URL, this is the simplest way.

Step 1: Create a GitHub Account (if you don’t have one)

  • Go to GitHub and sign up.

Step 2: Create a Repository

  1. Click the “+” (New Repository) button.
  2. Name it something like blackrock-token.
  3. Set it to public.
  4. Check "Add a README" and click "Create Repository".

Step 3: Upload the Website Files

  1. Click “Add File” → “Upload Files”.
  2. Upload the HTML file I provided.
  3. If you have a logo, upload it as logo.png.
  4. Click “Commit Changes”.

Step 4: Enable GitHub Pages

  1. Go to Settings → Pages.
  2. Under "Branch", select "main" and click Save.
  3. Your site will be live at:
    https://yourusername.github.io/blackrock-token/

Option 2: Netlify (Custom Domain + Easier Updates)

Netlify gives you a custom subdomain and easy deployment.

Step 1: Create a Netlify Account

  • Sign up at Netlify using GitHub, Google, or Email.

Step 2: Upload Your Website

  1. Click “New Site from Git” (or drag the HTML file into Netlify for manual upload).
  2. Connect to GitHub and select your blackrock-token repo.
  3. Click “Deploy Site”.

Step 3: Get Your Free Subdomain

  • After deployment, go to Site Settings → Domain Management.
  • Click “Edit” and set a custom subdomain like:
    blackrock-crypto.netlify.app.

Bonus: Get a Custom Domain (Optional)

If you want a .com or .xyz domain:

  1. Buy a domain on Namecheap or Google Domains.
  2. Set Netlify or GitHub Pages as the host.
